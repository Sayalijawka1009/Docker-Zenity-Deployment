# Docker-Zenity-Deployment
GUI-based Docker Image Deployment Automation Using Bash and Zenity
Streamlined Docker Image Push with Zenity is a DevOps automation project that simplifies the Docker image deployment process through a user-friendly GUI. Using Shell Scripting, Docker, and Zenity, the application allows users to log in to Docker Hub, select a Dockerfile, build and tag Docker images, and push them to a repository with minimal manual effort. This project enhances deployment efficiency, reduces command-line complexity, and provides a seamless experience for managing Docker images.
